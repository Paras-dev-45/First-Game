# Endless Runner Game

An original HTML5 endless runner game inspired by the genre of Subway Surfers.
Built with pure HTML5 Canvas, CSS3, and Vanilla JavaScript — no libraries or frameworks.

## Folder Structure

```
Endless Runner/
│
├── index.html              ← Main entry point of the game
├── style.css               ← Visual styling and layout of the game UI
├── game.js                 ← Main game engine and logic
│
└── assets/
    ├── images/             ← Sprite sheets, backgrounds, character art
    ├── sounds/             ← Sound effects (jump, coin, collision, etc.)
    └── fonts/              ← Custom game fonts
```

## How to Play

- Open `index.html` in your browser
- Press **SPACE** or **SWIPE UP** to jump
- Press **LEFT/RIGHT ARROW** or **SWIPE LEFT/RIGHT** to switch lanes
- Avoid obstacles and collect coins
- Survive as long as possible and beat your high score!
